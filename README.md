# 🐍 Basilisk

### the autonomous AI pentester that proves every exploit

<a href="https://github.com/the-priest/Basilisk"><img src="https://img.shields.io/badge/CODE-github.com%2Fthe--priest%2FBasilisk-e11d2b?style=for-the-badge&logo=github&logoColor=white&labelColor=08090b" alt="Repository"/></a>
<a href="https://the-priest.github.io/Basilisk/"><img src="https://img.shields.io/badge/LIVE-SITE-7d121b?style=for-the-badge&logo=githubpages&logoColor=white&labelColor=08090b" alt="Live site"/></a>

</div>

A fully autonomous, black-box AI penetration-testing agent that runs as a **desktop app (GTK4 / libadwaita)** on your own Kali Linux box — with real hands on the machine through tools. It plans, attacks, and **verifies every exploit with an out-of-band oracle** before it claims a win. No hallucinated findings. If Basilisk says it popped something, it *proved* it.
